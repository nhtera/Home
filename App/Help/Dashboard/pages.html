Page Heirarchy System

Your website is made up of pages and works similar to a file system. Some pages may act like folders
on a file system, containing a set of sub-pages.

Sub Pages

You can create sub-pages for almost any page within your website. Pages that include a folder icon
will contain one or more sub-pages. You can navigate to a list of sub-pages by clicking on a page's folder icon.

The URL for a sub-page will look something like yoursite.com/blog/sub-page

Page Types

There are two page types, Clone & Shadow. Depending on the type of page you are working with, your sub-pages will behave differently.
The Clone page type will include a special sub-page called a template page.

Creating a new sub-page will simply duplicate the contents within the template page.
If you decide to change the contents of the template page, it may not affect any cloned sub-pages, unless the content you are changing belongs to a layer that exists on both the template page & on your sub-pages.

The Shadow page type will include a special sub-page called a template page.

All sub-pages will load as a "shadow" of the template page. If you add, rearrange or remove any content from the template page, it will affect all sub-pages because they share the same content. You can modify the content of your sub-pages to override the "shadowed" content from the template page, but you can only add, rearrange, or remove content from the template page.
